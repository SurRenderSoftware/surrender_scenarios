# How to

1. Install python requirements to execute User Manual scripts
```
cd <surrender installation path>/scripts/user_manual
pip install -r requirements.txt
pip install <surrender installation path>/interfaces/python/surrender
```

2. Start SurRender Server
```
cd <surrender installation path>
sh start_server_headless.sh
```

3. Execute User Manual script
```
cd <surrender installation path>/scripts/user_manual
python script_XX_....py
```
